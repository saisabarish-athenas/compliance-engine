## Column Alignment

```
Column 1:  No. in register          → CENTER
Column 2:  Name                     → LEFT
Column 3:  Department               → LEFT
Column 4:  Dates worked             → CENTER (empty)
Column 5:  Extent of overtime       → RIGHT (2 decimals)
Column 6:  Piece-worker OT          → RIGHT (2 decimals)
Column 7:  Normal hours             → RIGHT (fixed 8)
Column 8:  Normal rate of pay       → RIGHT (2 decimals)
Column 9:  Overtime rate of pay     → RIGHT (2 decimals)
Column 10: Normal earnings          → RIGHT (2 decimals)
Column 11: Overtime earnings        → RIGHT (2 decimals)
Column 12: Food grain benefit       → RIGHT (2 decimals)
Column 13: Total earnings           → RIGHT (2 decimals)
Column 14: Payment dates            → CENTER (empty)
```

## Data Row Example

```
Input Data:
- employee_name: "John Doe"
- designation: "Manufacturing"
- overtime_hours: 5.00
- basic_salary: 10,000
- is_piece_worker: true
- food_grain_benefit: 0

Calculated Values:
- daily_rate: 10,000 / 26 = 384.62
- hourly_rate: 384.62 / 8 = 48.08
- overtime_rate: 48.08 × 2 = 96.15
- normal_earnings: 384.62 × 1 = 384.62
- overtime_earnings: 96.15 × 5 = 480.75
- piece_worker_overtime: 5.00 (because is_piece_worker = true)
- total_earnings: 384.62 + 480.75 + 0 = 865.37

Rendered Row:
│ 1  │John Doe│Manufacturing│          │   5.00   │   5.00   │   8    │ 48.08  │ 96.15  │ 384.62 │ 480.75 │   0.00   │ 865.37 │          │
```

## Totals Row Example

```
Input Data (3 employees):
- Employee 1: OT 5.00, Normal Rate 48.08, OT Rate 96.15, Normal Earnings 384.62, OT Earnings 480.75
- Employee 2: OT 3.00, Normal Rate 48.08, OT Rate 96.15, Normal Earnings 384.62, OT Earnings 288.45
- Employee 3: OT 4.50, Normal Rate 48.08, OT Rate 96.15, Normal Earnings 384.62, OT Earnings 432.68

Calculated Totals:
- Total OT Hours: 5.00 + 3.00 + 4.50 = 12.50
- Total Piece-Worker OT: 5.00 + 0 + 0 = 5.00
- Total Normal Rate: 48.08 + 48.08 + 48.08 = 144.24
- Total OT Rate: 96.15 + 96.15 + 96.15 = 288.45
- Total Normal Earnings: 384.62 + 384.62 + 384.62 = 1,153.86
- Total OT Earnings: 480.75 + 288.45 + 432.68 = 1,201.88
- Total Food Grain: 0 + 0 + 0 = 0.00
- Total Earnings: 1,153.86 + 1,201.88 + 0 = 2,355.74

Rendered Row:
│    │TOTAL   │        │          │  12.50   │   5.00   │        │144.24  │288.45  │1153.86 │1201.88 │   0.00   │2355.74 │          │
```
